## MUPhone Client macOS v0.1.0 — Initial Release

### Features
- 12+ device grid view with real-time H.264 streaming
- ShortcutBar with configurable quick-launch buttons
- HD/FHD quality switching with server-side profile persistence
- Multi-client support (shared/exclusive control modes)
- Manual clipboard sync (Ctrl+V to active device)
- Custom close confirmation dialog
- Configurable settings shortcut key
- Independent device windows with alias titles
- Auto-reconnect with exponential backoff
- 38 Material icons for shortcuts

### Notes
- macOS build is compiled via GitHub Actions
